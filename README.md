# Memory Card

Memory card game create with React.

[Live Demo](https://victavila.github.io/memory-card/)

## Getting started
```
git clone https://github.com/victavil/memory-card.git
cd memory-game
npm install
npm start
```

## Built with
- [React](https://reactjs.org/)
- [jikanjs](https://github.com/zuritor/jikanjs/)
